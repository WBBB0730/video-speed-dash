# Video Dash - 轻量级视频调速助手

### 主要功能
- [x] 按住快捷键实时调整视频播放速度(支持0.1x-16x)
- [x] 松开按键立即恢复原始播放速度
- [x] 为不同快捷键绑定个性化倍速配置
- [x] 支持所有主流视频网站

### 为什么选择 Video Dash
- 高效场景：快速过教程/慢放细节/跳过冗长片段
- 轻量易用：无需复杂设置，安装后自动生效
- 隐私承诺：零数据收集，完全本地化运行
- 开源免费：MIT协议开源，支持自定义扩展

适合学生、内容创作者及需要精准控制视频节奏的用户。

### 开发

```bash
pnpm i
```

```bash
pnpm dev
```

然后**在浏览器中加载 `extension/` 文件夹作为扩展**。

### 构建

```bash
pnpm build
```

然后打包 `extension` 下的文件，将 `extension.zip` 上传到相应的扩展商店。
